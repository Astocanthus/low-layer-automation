## [0.1.0] - 2026-02-01

### Added

- **Project Foundation** (`low-layer/`)
  - Initial repository structure for meta/documentation
  - LOW-LAYER Source Available License v1.0
  - Multi-repo architecture documentation

- **LICENSE** (`LICENSE`)
  - Custom Source Available license
  - Free for up to 5 users
  - Commercial license required for larger teams
  - SaaS competitor restriction

- **Architecture Documentation** (`ARCHITECTURE.md`)
  - Complete multi-repository structure definition
  - 7 repositories: api, cli, console, terraform-provider, operators, charts
  - Inter-repository dependencies mapping
  - Tech stack specifications (Go, TypeScript, React)
  - Development workflow guidelines

- **Coding Standards** (`CLAUDE.md`)
  - Code generation standards with mandatory headers
  - README standards for all repositories
  - Changelog standards (Keep a Changelog format)
  - Git workflow (version branches, feature branches)
  - Security hook configuration

- **README** (`README.md`)
  - Platform overview and features
  - Quick start guides (CLI and Terraform)
  - Service catalog listing
  - Architecture diagram
